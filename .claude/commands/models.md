# 模型选择指南

显示 Claude 模型与智谱 GLM 模型的对应关系，帮助用户选择合适的模型。

## 智谱 API 模型映射

当前使用智谱 API（`api.z.ai`），Claude 模型 ID 与智谱 GLM 模型的对应关系：

| Claude 模型 ID | 智谱模型 | 说明 | 适用场景 |
|---------------|---------|------|---------|
| `claude-opus-4-8` | glm-5.2 | 旗舰级，最强推理 | 复杂架构决策、深度分析、困难调试 |
| `claude-sonnet-5` | glm-5.1 / glm-5-turbo | 均衡型，性价比高 | 日常开发、代码编写、多智能体协作 |
| `claude-haiku-4-5` | glm-4.5-air | 轻量快速，成本低 | 简单任务、快速问答、轻量代理 |

## 切换模型

请使用内置 `/model` 命令切换模型，选择对应的 Claude 模型 ID 即可，智谱会自动映射到对应的 GLM 模型。
